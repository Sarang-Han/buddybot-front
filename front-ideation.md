## 메인 UI 핵심 구성요소

### 히어로 섹션 (Hero Section)
- 챗봇 캐릭터 버디의 인사
- 프로필 이미지 활용

### 사용자 정보 기입 카드 (미니 학생증) 

- 학년 / 전공 / 이수 학기(학년) 선택
- 채팅 입력전 필수로 기입해야함


### "지금 중요해!" - 시기별 알림 배너

```
┌─────────────────────────────────────────┐
│  🔥 지금 꼭 확인해야 할 것들!                │
│                                         │
│  📅 2월 20일 - 2월 24일                │
│  • 수강신청 확인 및 변경               │
│  • 복학 신청 마감 (2/21)               │
└─────────────────────────────────────────┘
```

**데이터 소스**:
- `academic_calendar.csv`에서 오늘 날짜 기준 ±7일 이벤트 자동 추출
- 이벤트 없으면 기본 문구
- `public/data/academic_calendar.csv` 링크를 참조할것.

### "학년별 맞춤 정보" - 퍼스널라이제이션 섹션

```
┌─────────────────────────────────────────┐
│   👤 [학년 선택: 1학년 ▼]              │
│                                         │
│   1학년 신입생이라면 이것부터!          │
│                                         │
│   ✅ 수강신청 처음이지? → [도움말]     │
│   ✅ 채플 꼭 들어야 해 → [자세히]      │
│   ✅ 전공 결정은 2학년 때 → [미리보기] │
│   ✅ 복수전공 고민된다면 → [알아보기]  │
│                                         │
│   💡 [학년별 가이드 전체보기]          │
└─────────────────────────────────────────┘
```

- 드롭다운에서 학년 선택 → 맞춤 콘텐츠 변경
- 또는 최초 방문 시 "학년 선택" 모달 표시

```
# 내부 컨텐츠 예시

1학년
- 나의 첫 수강신청 준비하기
- 나의 첫 기숙사 입사 준비
- 학교생활 필수 세팅 (sw지원, 시설, 학생활동 등)
- 교내 커뮤니티 100% 활용하기 (에브리타임, 이화이언)
- 채플(훈련학점)이 무엇인가요?
- 필수교양 핵심교양? (수강신청 안내 관련)
- 학기 중 꼭 챙겨야 할 일정 미리보기 (학사일정)

2학년
- 복수전공/부전공 고민된다면
- 교환학생/방문학생
- ... (채워주세요)

3학년
- ... (채워주세요)

4학년
- ... (채워주세요)
```
